# Lerny Landing Page · דף הנחיתה של Lerny

דף נחיתה שיווקי עצמאי (standalone) עבור **Lerny** — מערכת ניהול בית הספר.
הפרויקט נפרד לחלוטין מקוד האפליקציה: HTML / CSS / JS סטטיים, ללא build, ללא תלויות.

## הרצה מקומית

```bash
cd landing
python3 -m http.server 8080
# ואז לפתוח http://localhost:8080
```

או פשוט לפתוח את `index.html` בדפדפן.

## פריסה

כל אחסון סטטי עובד כמו שהוא (Vercel / Netlify / Cloudflare Pages / GitHub Pages) —
מפנים את ה-root לתיקיית `landing/`.

## מבנה

| קובץ | תפקיד |
|---|---|
| `index.html` | כל תוכן הדף — Hero, יכולות, תפקידים, הצצה למערכת, איך מתחילים, CTA |
| `styles.css` | עיצוב מלא, כולל טוקני המותג של Lerny (Heebo, primary blue, radius) ורספונסיביות |
| `script.js` | אינטראקציות: תפריט מובייל, reveal בגלילה, מוני סטטיסטיקה, טאבים לפי תפקיד, tilt תלת־ממדי, סרגל התקדמות, טופס CTA |

## הערות מותג

- צבע ראשי: `oklch(0.55 0.18 252)` — זהה לאפליקציה
- פונט: Heebo (Google Fonts), משקלים 400–900
- לוגו: GraduationCap על ריבוע כחול מעוגל — תואם ל-app shell
- כל הדף RTL בעברית, כולל המוקאפים של המערכת (בנויים ב-HTML/CSS, לא צילומי מסך)

## מה עוד לא מחובר

- טופס "קבעו הדגמה" מציג אישור מקומי בלבד — צריך לחבר ל-backend / CRM / שירות טפסים
- קישורי "תנאי שימוש" / "מדיניות פרטיות" בפוטר הם placeholders
